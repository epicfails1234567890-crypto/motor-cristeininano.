# motor-cristeininano.

El motor presente se mueve a la par q se aplica la secuencia rotativa sobre las bobinas con arduino u otra placa preparada para el sometimiento de dichas fuerzas accelerativas al estator, mientras q el rotor magnético gira suspendido en una camara de vacío con imanes q no aparecen en la imagen q a su vez hacen q el rotor no se salga de su sitio.
